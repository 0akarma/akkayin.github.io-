Act as an Amateur OPS in D0g3-CTFd

Study
  • Docker
  • Linux
  • Summary

学校的服务器一直没给我们安置好，只好用自己的vps给学弟学妹们搭靶场环境了～～于是乎就当了回业余运维。

前言

用vps自然不能像学校服务器那样：疯狂开虚拟机搭环境；小水管只可以用docker来苟活:(

Docker

制作一个模版image

为了减少重复的无聊操作，可以先创建一个容器，在里面装好所需的工具与环境（vim,ssh……），再commit成一个模版镜像，之后如果需要再开containers只需要再run一下就ok了。

这时候就可以分发docker给小伙伴们出题啦！！

###iptables-> docker 端口映射

当一个container里面搭建了多个端口的题目环境，需要映射到vps的公网ip时，有两种方法可以解决

重新创建一个container

  1. 先将现有container“备份”一下（commit）

  2. 然后再docker run一次，加上所需端口即可

    iptables 端口转发

用此方法需要稍微理解理解iptables的工作机制

维护

为了预防有人搅💩，而且懒得一挂掉，就要手动登上去调各种服务，所以还是写个脚本定时恢复好了～～
