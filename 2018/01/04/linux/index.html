Linux初探

Study
  • Linux
  • Notes
  • Summary

线代，网页制作都考完了，过几天才考高数，所以有几天空闲时间，就赶紧恶补一下linux吧！


常用命令

目录处理命令

ls 显示目录文件

-a 显示所有文件，包括隐藏文件

-l 详细信息展示 -h 人性化显示

-d 查看目录属性

-i

mkdir 创建新目录

-p 递归创建

cd 切换目录

.. ->当前目录的上一级目录

. ->当前目录

pwd 显示当前目录

rmdir 删除空目录

cp 复制

-f 复制目录

-p 保留文件属性

mv 剪切/改名

rm 删除

-rf [文件或目录]

-f 删除目录

-f 强制执行

文件处理命令

touch 创建空文件

cat 显示文件内容

-n 显示行号

tac 显示文件内容（反向）

more 分页显示文件内容

空格/f ->翻页

回车 ->换行

q或Q ->退出

less 分页显示文件内容（可向上翻页）

搜索：/关键词

head 显示文件前几行

-n 制定行数（默认10）

tail 显示文件后面几行

-n 指定行数

-f 动态显示文件末尾内容
