线代，网页制作都考完了，过几天才考高数，所以有几天空闲时间，就赶紧恶补一下linux吧！


文件处理命令

目录处理命令

ls 显示目录文件

grammar: ls [-ald]【文件或目录】

-a 显示所有文件，包括隐藏文件

-l 详细信息展示 -h 人性化显示

-d 查看目录属性

-i

mkdir 创建新目录

grammar: mkdir -p [目录名]

-p 递归创建

cd 切换目录

grammar: cd [目录名]

.. ->当前目录的上一级目录

. ->当前目录

pwd 显示当前目录

rmdir 删除空目录

grammar: rmdir [目录名]

cp 复制

grammar: cp [原文件或目录]【目标目录】

-f 复制目录

-p 保留文件属性

mv 剪切/改名

grammar: mv [原文件或目录]【目标目录】

rm 删除

grammar: rm [文件或目录]

-rf [文件或目录]

-f 删除目录

-f 强制执行


文件处理命令

touch 创建空文件

grammar: touch [文件名]

cat 显示文件内容

grammar: cat [文件名]

-n 显示行号

tac 显示文件内容（反向）

grammar: tac [文件名]

more 分页显示文件内容

grammar: more [文件名]

空格/f ->翻页

回车 ->换行

q或Q ->退出

less 分页显示文件内容（可向上翻页）

grammar: less [文件名]

搜索：/关键词

head 显示文件前几行

grammar: head [文件名]

-n 制定行数（默认10）

tail 显示文件后面几行

grammar: tail [文件名]

-n 指定行数

-f 动态显示文件末尾内容


链接命令

ln 生成链接文件

软链接：

grammar: ln -s [原文件]【目标文件】

类似windows的快捷方式

-s 创建软链接

1.lrwxrwxrwx l ->软链接

2.文件很小

3.箭头指向源文件

硬链接：

grammar: ls [原文件]【目标文件】

除了位置不一样，其余都一样（像cp -p 一样）

  1. cp -p + 同步更新

    echo “123456” >>/etc/issue

  2. 通过 i 节点识别

  3. 不能跨分区

  4. 不能针对目录使用

权限管理命令

权限的数字表示

r —- 4

w —- 2

x —- 1

rwxrw-r–

7 6 4

chmod 改变文件或目录权限

grammar: chgmod [{ugoa} {+-=} [文件或目录]

-R 递归修改

代表字符 权限 对文件的含义 对目录的含义
r 读权限 可以查看文件内容(cat/more/head/tail……) 可以列出目录中的内容(ls)\
w 写权限 可以修改文件内容(vim) 可以在目录中创建、删除文件(touch/mkdir/rmdir/rm)
x 执行权限 可以执行文件(script/command) 可以进入目录(cd)\

chown 改变文件或目录的所有者

grammar: chown [用户]【文件或目录】

chgrp 改变文件或目录所属组

grammar: chgrp [用户组]【文件或目录】

umask 显示、设置文件的缺省权限

grammar: umask [-S]

-S 以rwx形式显示新建文件的缺省权限

umask 022 -> 755

文件搜索命令

find 文件搜索

grammar: find [搜索范围]【匹配条件】

-name [name]

(-iname 不区分大小写)

-size [+/- ]

一个数据块 512字节 0.5k

-user

-group

-cmin -5

文件属性 change

-amin -5

访问时间 access

-mmin -5

文件内容 modify

-exec/ok cmd {} \;

对搜索结果执行命令

-type 根据文件类型查找

f 文件 d 目录 l 软链接文件

-inum 根据i节点查找

-a 两个条件同时满足 -o 两个条件满足一个即可

eg: find /etc -size +163840 -a -size -204800

locate 在文件资料库中查找文件

grammar: locate [文件名]

updatedb

which 搜索命令所在目录及别名信息

grammar: which [命令]

whereis 搜索命令所在目录及帮助文档路径

grammar: whereis [命令]

grep 在文件中搜寻字符串匹配的行并输出

grammar: grep [指定字符串]【文件】

-i 不区分大小写

-v 排除指定字符串 (^# -> 行首#)
